# dux

フォルダ単位でディスク使用量を再帰的に集計・表示するGo製CLIツール。
不要な大容量ファイルの整理に。

## 特徴

- **高速スキャン** — ゴルーチンによる並列スキャン（`-j` でワーカー数調整）
- **インタラクティブTUI** — ターミナルで実行するとツリーを展開・折りたたみながら探索できる
- **テキスト出力** — パイプやリダイレクトにも対応
- **サイズフィルタ** — 指定サイズ未満のフォルダを非表示
- **実ディスク使用量** — 10GB以上のファイルは `GetCompressedFileSizeW` で実際のディスク割り当てサイズも取得（仮想ディスク等の圧縮・スパースファイルに対応）

## インストール

```
go install github.com/s-hosoai/dux@latest
```

## 使い方

```
dux [オプション] [パス]
```

パスを省略すると現在のディレクトリをスキャンします。

### オプション

| フラグ | デフォルト | 説明 |
|---|---|---|
| `-j N` | CPU数 | 並列スキャン数（SSDでは増やすと高速化） |
| `-min SIZE` | `0` | 最小サイズフィルタ（例: `100MB`, `1GB`） |
| `-name` | — | 名前順ソート（デフォルト: サイズ降順） |
| `-flat` | — | フラットリスト表示（TUI無効） |
| `-no-color` | — | カラー出力を無効化（TUI無効） |
| `-depth N` | `-1`（無制限） | 最大表示深さ（テキストモード） |
| `-top N` | `0`（全表示） | 表示件数の上限（テキストモード） |

### 例

```powershell
# カレントディレクトリをTUIで表示
dux .

# Cドライブを100MB以上のフォルダのみ表示
dux -min 100MB C:\

# 上位20件をフラット表示
dux -flat -top 20 C:\Users

# ファイルにリダイレクト
dux -no-color C:\Users > result.txt
```

## TUI操作

ターミナルで実行すると自動的にTUIモードになります。

| キー | 動作 |
|---|---|
| `↑` / `k` | 上に移動 |
| `↓` / `j` | 下に移動 |
| `Enter` / `Space` / `→` / `l` | ツリーを展開・折りたたみ |
| `←` / `h` | 折りたたむ（折りたたみ済みの場合は親へ移動して折りたたむ） |
| `q` / `Esc` | 終了 |

```
  ▼    1.23 TB  C:\Users\example
> ▼  890.12 GB    ▼ Videos
    123.45 GB      ▶ Archive
     45.67 GB      ▶ Projects
  ▶   98.76 GB    ▶ Documents
      12.34 GB    ▶ Downloads

   890.12 GB  C:\Users\example\Videos
  ↑↓/jk: 移動   Enter/Space/→/l: 展開   ←/h: 折りたたみ   q: 終了
```
